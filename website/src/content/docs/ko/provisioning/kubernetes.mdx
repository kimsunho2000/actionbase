카카오에서 Actionbase는 현재 Kubernetes를 사용하여 프로비저닝되고 있습니다. 이 문서에는 배포 매니페스트, 고가용성 구성, 성능 튜닝, 모니터링 설정, 보안 모범 사례가 포함될 예정입니다.

- Actionbase를 체험하려면 [빠른 시작](/ko/quick-start/)을 참고하세요
- 기여하려면 [개발 환경 설정](/ko/community/development/)을 참고하세요
