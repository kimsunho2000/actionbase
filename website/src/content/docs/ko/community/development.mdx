## 사전 요구사항

- Java 17+
- IntelliJ IDEA (Community 또는 Ultimate)

## 설정

```bash
git clone https://github.com/kakao/actionbase.git
cd actionbase
```

IntelliJ에서 열기: **File > Open** → 프로젝트 루트 선택.

IntelliJ가 자동으로 Gradle을 가져옵니다. 인덱싱이 완료될 때까지 기다려 주세요.

## 빌드

```bash
./gradlew build
```

또는 IntelliJ에서: **Gradle panel > actionbase > Tasks > build > build**

## 포맷

커밋 전에 실행하세요:

```bash
./gradlew spotlessApply
```

프로젝트 스타일에 맞게 Kotlin/Java 코드를 포맷합니다.

## 실행

```bash
./gradlew :server:bootRun
```

서버가 `http://localhost:8080`에서 시작됩니다.

## 테스트

```bash
# 전체 테스트
./gradlew test

# 특정 모듈
./gradlew :core:test
./gradlew :engine:test
./gradlew :server:test
```

## PR 워크플로우

GitHub에서 [kakao/actionbase](https://github.com/kakao/actionbase)를 포크한 후 remote를 설정하세요:

```bash
git remote rename origin upstream
git remote add origin https://github.com/YOUR_USERNAME/actionbase.git
```

1. 브랜치 생성: `git checkout -b feature/your-feature`
2. 변경 사항 작성
3. 포맷: `./gradlew spotlessApply`
4. 테스트: `./gradlew test`
5. 커밋: `git commit -m "feat(scope): description"`
6. 푸시 및 PR 생성

커밋 메시지 형식과 CLA에 대해서는 [기여하기](/ko/community/contributing/)를 참고하세요.
