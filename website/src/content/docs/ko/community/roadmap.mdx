## 기능

- [ ] <Badge text="pub" variant="success" size="small" /> 실시간 집계
- [ ] <Badge text="new" variant="note" size="small" /> 제한된 멀티홉 쿼리
- [ ] <Badge text="new" variant="note" size="small" /> 실시간 top-N 쿼리 (리더보드)
- [ ] <Badge text="new" variant="note" size="small" /> 메타데이터 v2에서 v3으로
- [ ] <Badge text="new" variant="note" size="small" /> 메타스토어 통합

- [ ] <Badge text="new" variant="note" size="small" /> HBase와 Kafka 없는 경량 배포 (예:

## 계획 없음

- 범용 그래프 쿼리
- 제한 없는 순회 또는 분석
- 별도의 스토리지 엔진 구축
