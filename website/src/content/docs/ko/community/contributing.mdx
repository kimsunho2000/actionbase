Actionbase에 관심을 가져주셔서 감사합니다. 프로젝트 사용, 질문, 이슈 보고, 문서 개선, 코드 기여 등 어떤 형태의 참여든 감사드립니다.

## 협업 방식

[GitHub](https://github.com/kakao/actionbase)를 통해 협업합니다:

- **[Discussions](https://github.com/kakao/actionbase/discussions)**: 질문, 아이디어, 자유로운 대화
- **[Issues](https://github.com/kakao/actionbase/issues)**: 버그 보고, 기능 요청, 구체적인 개선 사항
- **[Pull Requests](https://github.com/kakao/actionbase/pulls)**: 코드 및 문서 변경

질문, 아이디어 또는 피드백이 있으시면 [Discussions](https://github.com/kakao/actionbase/discussions)에 참여해 주세요.

Pull request는 협력적으로 리뷰되며 Maintainer가 병합합니다.

Pull request를 제출할 때 Individual용 [CLA](https://cla-assistant.io/kakao/actionbase) (Contributor Licensing Agreement)에 서명해 주세요. Corporate용 Contributor Licensing Agreement가 필요한 경우 [문의해 주세요](mailto:oss@kakaocorp.com).

## 시작하기

오픈소스가 처음이신가요? **[good first issue](https://github.com/kakao/actionbase/labels/good%20first%20issue)** 레이블이 붙은 이슈를 찾아보세요.

로컬 개발 환경 설정은 [개발 환경 설정](/ko/community/development/)을 참고하세요.

## 번역

문서 번역 기여를 환영합니다. 번역 PR을 제출할 때는 한 번에 모든 페이지가 아닌 **섹션(폴더) 단위로 PR을 제출**해 주세요. 이렇게 하면 리뷰가 용이하고 점진적으로 진행할 수 있습니다.

## 보안

보안 취약점은 공개 이슈 대신 [GitHub Security Advisories](https://github.com/kakao/actionbase/security/advisories/new)를 통해 보고해 주세요.

## 커뮤니티 표준

모든 기여자는 **[행동 강령](https://github.com/kakao/actionbase/blob/main/CODE_OF_CONDUCT.md)**을 따라야 합니다.

프로젝트 역할에 대해서는 [거버넌스](/ko/community/governance/)를 참고하세요.
