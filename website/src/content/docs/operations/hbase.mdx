HBase configuration guides and best practices will be provided in future releases.

This is currently a real blocker for production adoption outside Kakao. Actionbase uses HBase as its primary storage backend, but we haven't yet documented the operational knowledge needed to run it reliably.

For now, see [Local Provisioning](/provisioning/local/) to run HBase locally for development and testing.
