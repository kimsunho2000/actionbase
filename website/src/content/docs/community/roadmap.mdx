## Features

- [ ] <Badge text="pub" variant="success" size="small" /> Real-time aggregation
- [ ] <Badge text="new" variant="note" size="small" /> Bounded multi-hop queries
- [ ] <Badge text="new" variant="note" size="small" /> Real-time top-N queries (leaderboards)
- [ ] <Badge text="new" variant="note" size="small" /> Metadata v2 to v3
- [ ] <Badge text="new" variant="note" size="small" /> Metastore consolidation

## Ecosystem

- [ ] <Badge text="pub" variant="success" size="small" /> Migration pipeline (Spark-based HBase bulk
      loading)
- [ ] <Badge text="pub" variant="success" size="small" /> Async processor (Spark Streaming,
      WAL-driven)
- [ ] <Badge text="pub" variant="success" size="small" /> Shadow testing (mirroring traffic to a
      test cluster)

## Production

- [ ] <Badge text="pub" variant="success" size="small" /> 1.0.0 release (production-ready for
      external users)
- [ ] <Badge text="pub" variant="success" size="small" /> Provisioning guide for Kubernetes
- [ ] <Badge text="pub" variant="success" size="small" /> Configuration reference
- [ ] <Badge text="pub" variant="success" size="small" /> Operations and monitoring guide
- [ ] <Badge text="pub" variant="success" size="small" /> Test-driven documentation (tests as
      contracts)
- [ ] <Badge text="pub" variant="success" size="small" /> Zero-downtime migration (legacy to
      Actionbase)
- [ ] <Badge text="new" variant="note" size="small" /> Zero-downtime migration (storage backend
      migration)

## Exploring

- [ ] <Badge text="new" variant="note" size="small" /> Lightweight deployment without HBase and
      Kafka (e.g., [SlateDB](https://github.com/slatedb/slatedb),
      [S2](https://github.com/s2-streamstore/s2))

## Not Planned

- General-purpose graph queries
- Unbounded traversal or analytics
- Building another storage engine
